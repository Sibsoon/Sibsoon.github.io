Normal Mapping (5 pt.)



Modified Files:
  • src/mesh.cpp
  • src/normal_map.cpp

The i/o handling is pretty much the same as it is for the texture mapping.
The NormalMap is part of the nori Texture interface. The normal map takes two optional arguments inspired by the checkerboard implementation, scale and delta. These optional parameters allow the user to shift and scale the texture if necessary.
In the setHitInformation of mesh.cpp is the implementation to adapt the mesh normals. It consists of a transformation to the tangent space. To evaluate the results we first compare the renderings with the normal integrator and then show a rendering with texture and normal mapping.

Problem

There is an issue with the tangent space transformation, also visible in the comparisons. The purple of the normal map image should be matching the blue-ish color of the un-mapped plane. This results in a darker end result, since there are more shadows rendered, as it should.
Unfortunately we could not find the bug.
